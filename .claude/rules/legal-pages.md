# Legal Page Rules

- Indonesian apps must reference: UU PDP No. 27/2022, PP No. 71/2019, Permenkominfo No. 20/2016
- Apps with children/minors must reference UU No. 35/2014 (Perlindungan Anak)
- Before editing legal docs, follow @.claude/skills/update-legal-docs/SKILL.md workflow
- Each project's legal docs live in `content/{project}/{lang}/`
- SIMSDIG (core): school is data controller, developer provides software only
