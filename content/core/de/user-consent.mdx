# Datenschutzoptionen der Nutzer für SIMSDIG

**Letzte Aktualisierung: 24. März 2026**

<h2 id="update-data">Ihre Daten aktualisieren</h2>

Sie können Ihre personenbezogenen Daten auf zwei Arten aktualisieren:

- **Über den Dienst:** Melden Sie sich bei SIMSDIG an und navigieren Sie zu Ihren Profileinstellungen, um Informationen wie E-Mail, Telefonnummer, Profilbild und andere bearbeitbare Felder zu aktualisieren.
- **Über die Schule:** Wenden Sie sich an das Verwaltungsbüro Ihrer Schule (Tata Usaha), um Änderungen an Daten zu beantragen, die nicht direkt über den Dienst geändert werden können, wie z. B. offizielle Identifikationsnummern (NIK, NISN, NIP).

---

<h2 id="deactivate-account">Ihr Konto deaktivieren</h2>

Benutzerkonten in SIMSDIG werden **von der Schulverwaltung erstellt und verwaltet** — nicht von den Nutzern innerhalb der Anwendung. Daher wird die Kontoverwaltung (einschließlich Deaktivierung) von der Schule durchgeführt.

Konten werden von der Schulverwaltung auf Grundlage des akademischen Status deaktiviert:

- **Abschluss:** Ihre Kontorolle wird auf **Alumni** geändert, mit eingeschränktem Zugang zum Verbleibsstudienprogramm (*program Kemendikti*).
- **Wechsel oder Schulverweis:** Ihr Konto wird vollständig deaktiviert.
- **Freiwilliger Antrag:** Sie können die Deaktivierung über das Verwaltungsbüro der Schule (Tata Usaha) beantragen. Beachten Sie, dass dies als Austritt aus der Schule behandelt wird.

Nach der Deaktivierung:

- Ihre Anmeldedaten werden deaktiviert.
- Sie können nicht mehr auf den Dienst zugreifen.
- Ihr Profil ist für andere Nutzer nicht mehr sichtbar.

**Hinweis:** Die Kontodeaktivierung führt nicht zur dauerhaften Löschung von Daten. Alle Bildungsunterlagen werden gemäß den indonesischen nationalen Bildungsvorschriften aufbewahrt. Siehe [Einschränkungen bei der Datenlöschung](#data-deletion-limits) weiter unten.

---

<h2 id="data-deletion-limits">Einschränkungen bei der Datenlöschung</h2>

Bestimmte Aufzeichnungen sind Teil des **nationalen Bildungsdatensystems (Dapodik)**, das vom indonesischen Bildungsministerium verwaltet wird. Aufgrund regulatorischer Anforderungen können die folgenden Daten **nicht dauerhaft gelöscht** werden:

- NISN (Nationale Schüler-ID)
- NIP (Mitarbeiternummer)
- Einschreibungsunterlagen und akademischer Verlauf
- Andere für die Dapodik-Konformität erforderliche Daten

Das Löschen dieser Daten würde die Integrität der nationalen Bildungsdaten beeinträchtigen. Die Schule bewahrt als Datenverantwortlicher diese Daten gemäß den indonesischen Bildungsvorschriften auf.

<h2 id="revoke-permissions">Berechtigungen widerrufen</h2>

Sie können Geräteberechtigungen jederzeit über die Systemeinstellungen Ihres Geräts widerrufen:

- **Kamera:** Wird für Anwesenheitsfotos und Profilbildaufnahme verwendet. Widerrufen über Einstellungen → SIMSDIG → Kamera.
- **Standort:** Wird für das GPS-basierte Ein-/Ausstempeln verwendet. Widerrufen über Einstellungen → SIMSDIG → Standort.
- **Fotobibliothek:** Wird verwendet, um ein vorhandenes Foto bei der Aktualisierung Ihres Profilbilds auszuwählen. Widerrufen über Einstellungen → SIMSDIG → Fotos.

Das Widerrufen von Berechtigungen kann bestimmte Funktionen des Dienstes einschränken (z. B. funktioniert die GPS-Anwesenheit ohne Standortzugriff nicht, die gesichtsbasierte Anwesenheit funktioniert ohne Kamerazugriff nicht).

<h2 id="data-portability">Datenübertragbarkeit</h2>

Wenn Sie eine Kopie Ihrer im Dienst gespeicherten personenbezogenen Daten erhalten möchten, wenden Sie sich bitte an das Verwaltungsbüro Ihrer Schule (Tata Usaha). Die Schule kann als Datenverantwortlicher Ihre Daten auf Anfrage in einem gängigen elektronischen Format (wie PDF oder Tabellenkalkulation) bereitstellen, gemäß Ihren Rechten nach <a href="https://peraturan.bpk.go.id/Details/229798/uu-no-27-tahun-2022" target="_blank" rel="noopener noreferrer">UU PDP No. 27/2022 (Datenschutzgesetz)</a>.

Sie haben außerdem das Recht, die Übertragung Ihrer Daten an eine andere Organisation in einem strukturierten, maschinenlesbaren Format zu verlangen, vorbehaltlich der technischen Machbarkeit und der geltenden Vorschriften.
